Introduction

Hospitals face challenges in managing costs and resources while ensuring quality care. This project showcases my ability to deliver data-driven solutions for healthcare efficiency.

Project Overview

This project analyzed patient data to uncover trends in costs, conditions, and demographics, driving a 20% efficiency gain. I used SQL for data extraction, Python for preprocessing, and Power BI for interactive visualizations, delivering actionable insights to stakeholders.

Objectives

  • Identify high-risk patient groups by demographics.
  • Analyze prevalent medical conditions by age.
  • Optimize facility utilization through patient distribution.
  • Evaluate billing patterns to reduce costs.
  • Assess length of stay for resource planning.

Dataset Description

Sourced from Kaggle, the dataset includes 50,000+ records with 15 attributes, covering patient admissions, costs, and outcomes.

  • Name: Patient’s full name
  • Age: Age at admission
  • Gender: Patient gender
  • Medical Condition: Primary diagnosis
  • Billing Amount: Treatment cost
  • Admission Type: Emergency, planned
  • Discharge Date: Discharge date
  • Blood Type: E.g., A+, O-
  • Doctor: Assigned physician
  • Hospital: Facility name
  • Insurance Provider: Insurer
  • Room Number: Assigned room
  • Medication: Prescribed drugs
  • Test Results: Diagnostic outcomes

Data Cleaning

I cleaned the dataset using SQL and Python (Pandas), removing 5% duplicates and standardizing formats for accurate analysis.

  • Removed 2,500 duplicates with SQL DISTINCT.
  • Standardized names using Python’s `str.title()`.
  • Handled missing values with imputation.
  • Verified data integrity post-cleaning.
